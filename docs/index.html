Field Assistant is a no-bullshit data collection app for researchers.

Key Features:

  • Offline capable
  • Record collection date-time
  • Record GPS position
  • Attach photos
  • Data sets and custom fields
  • CSV export (bundled zip)

The app is currently available as a PWA (with a limited feature set) at:
bugman-staging.firebaseapp.com

The original prototype achieved a great trial run with it's minimum features. This expanded into some experimental features for data sets in a beta trial.

Although fully capable, the web app environment is restrictive, especially on iOS. This led to a native approach using Qt/QML. Soon to be released, the full native version features photo support, data sets, and bundled zip exports.
